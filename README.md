# PharmaLearn — Digital Pharma Analytics Library

A world-class digital learning platform for pharma analytics professionals.

## Live Demo
Hosted on GitHub Pages: https://[your-username].github.io/pharmalearn

## Run Locally
```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

## Features
- 5 domains, 31 chapters (1 fully available, rest coming soon)
- Full Chapter 2.1: Patient Journey Analytics in Oncology
- Interactive knowledge checks with instant feedback
- Progress tracking (localStorage)
- Bookmarking system
- Dark/Light mode
- Search by topic or tag
- XP system and level progression
- Onboarding flow with role/geography personalization

## Stack
- HTML5 + Tailwind CSS (CDN)
- Alpine.js (CDN) for reactivity
- Prism.js for code highlighting
- No build step required
